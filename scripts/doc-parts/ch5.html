5.2 واجهات المستخدم الرئيسية

تم تصميم واجهات المنصة بأسلوب Dark Theme احترافي يعكس طابع الأمن السيبراني، مع دعم كامل لـ RTL:

الصفحة الرئيسية

تعرض خريطة هجمات سيبرانية تفاعلية ثلاثية الأبعاد (Globe)، إحصائيات المنصة، وأحدث الدورات والمختبرات. تم بناؤها باستخدام WebGL لتقديم تجربة بصرية مؤثرة.

لوحة تحكم الطالب (Student Dashboard)

تعرض: الدورات المسجلة مع نسبة الإتمام، الشارات المكتسبة (Badges)، النقاط، المختبرات المكتملة، والإشعارات.

واجهة المختبر التفاعلي (Lab Interface)

تحتوي على: وصف السيناريو، طرفية أوامر (Terminal)، حقل إدخال الـ Flag، مؤشر التقدم، والتلميحات.

لوحة تحكم المدير (Admin Dashboard)

تشمل: إدارة المستخدمين، إدارة الدورات والمحتوى، نظام المستندات (DMS)، سجلات التدقيق الأمني، وتقارير الأداء.

// Middleware: Authentication Guard
export async function getAuthUser(req: Request) {
  const token = req.headers.get('cookie')
    ?.split(';')
    .find(c => c.trim().startsWith('session='))
    ?.split('=')[1];
    
  if (!token) return null;
  
  const session = await db.session.findUnique({
    where: { token },
    include: { user: true },
  });
  
  if (!session || session.expiresAt < new Date()) return null;
  return session.user;
}
