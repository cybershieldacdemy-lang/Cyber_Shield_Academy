الفصل الأول: المقدمة العامة

1.1 تعريف المشروع

مشروع CyberShield Academy هو منصة تعليمية متكاملة متخصصة في مجال الأمن السيبراني، تهدف إلى تقديم تجربة تعليمية تفاعلية شاملة تجمع بين المحتوى النظري (دورات، فيديوهات، مقالات) والتطبيق العملي (مختبرات افتراضية، تحديات CTF، اختبارات تقييمية) مع نظام شهادات معتمد ونظام تقدم ذكي يتتبع مستوى الطالب.

تم بناء المنصة باستخدام تقنيات حديثة تشمل Next.js 16 كإطار عمل للواجهة الأمامية والخلفية (Full-Stack)، وPrisma ORM لإدارة قاعدة البيانات SQLite، مع تطبيق أفضل ممارسات أمن المعلومات في جميع طبقات التطبيق بما يشمل المصادقة الثنائية (2FA)، تشفير كلمات المرور عبر bcrypt، وحماية واجهات API.

الرؤية: بناء أكبر منصة عربية متخصصة في تعليم الأمن السيبراني، تدمج بين التعليم الأكاديمي والتدريب العملي المبني على سيناريوهات واقعية من عالم الاختراق والحماية.

1.2 أهمية التعليم في الأمن السيبراني

يشهد العالم تصاعداً غير مسبوق في الهجمات السيبرانية، حيث أظهرت تقارير IBM Security X-Force 2024 ارتفاع تكاليف اختراقات البيانات إلى معدل 4.88 مليون دولار لكل حادثة. وفي المنطقة العربية تحديداً، أشار تقرير المركز الوطني للأمن السيبراني (NCA) إلى نقص حاد في الكوادر المتخصصة يقدر بأكثر من 30,000 متخصص في السوق السعودي وحده.

تبرز أهمية هذا المشروع في النقاط التالية:

  • سد الفجوة في المحتوى العربي: أكثر من 95% من محتوى الأمن السيبراني المتقدم متوفر باللغة الإنجليزية فقط.
  • التعلم بالممارسة (Learning by Doing): توفير مختبرات افتراضية تحاكي بيئات حقيقية لتطبيق المهارات.
  • خفض تكلفة التدريب: بدائل فعّالة اقتصادياً مقارنة بالتدريب التقليدي الذي قد يكلف آلاف الدولارات.
  • تلبية متطلبات سوق العمل: مواءمة المناهج مع الشهادات المهنية المعتمدة مثل CEH، OSCP، CompTIA Security+.

1.3 أهداف المشروع

الأهداف الرئيسية:

  1. تصميم وتطوير منصة تعليمية متكاملة باللغة العربية والإنجليزية متخصصة في الأمن السيبراني.
  2. توفير مختبرات افتراضية تفاعلية (Cyber Labs) تحاكي سيناريوهات اختراق وحماية حقيقية.
  3. بناء نظام اختبارات وتقييم ذكي مع بنك أسئلة متعدد المستويات.
  4. تطوير نظام شهادات رقمية قابلة للتحقق عبر QR Code.
  5. تطبيق أعلى معايير الأمان في بنية المنصة نفسها كنموذج عملي.

الأهداف الفرعية:

  • نظام مدربين متكامل مع لوحة تحكم مخصصة.
  • نظام إدارة مستندات (DMS) داخلي لتوثيق العمليات الإدارية.
  • لوحة مراقبة أمنية (Security Dashboard) لتتبع الحوادث والثغرات.
  • خريطة تفاعلية للهجمات السيبرانية في الوقت الحقيقي.

الفصل الثاني: الدراسات السابقة وتحليل الأنظمة

2.1 منصات تعليمية عامة

2.1.1 منصة Udemy

تعتبر Udemy من أكبر منصات التعلم الإلكتروني عالمياً بأكثر من 210 مليون مستخدم و213,000 دورة. تعتمد نموذج سوق مفتوح (Marketplace) حيث يمكن لأي شخص نشر دورة. تقدم محتوى عام في مختلف المجالات بما فيها الأمن السيبراني.

نقاط القوة: مكتبة ضخمة، أسعار تنافسية، واجهة سهلة. نقاط الضعف: لا تحتوي على مختبرات عملية، جودة المحتوى متفاوتة، شهادات غير معتمدة مهنياً.

2.1.2 منصة Coursera

تتعاون Coursera مع جامعات مرموقة مثل Stanford وGoogle لتقديم برامج أكاديمية. توفر تخصصات في الأمن السيبراني مثل Google Cybersecurity Certificate.

نقاط القوة: شراكات أكاديمية، شهادات معترفة. نقاط الضعف: محتوى نظري بشكل كبير، تكلفة عالية، محتوى عربي محدود جداً.

2.2 منصات الأمن السيبراني المتخصصة

2.2.1 منصة TryHackMe

منصة بريطانية متخصصة في تعليم الأمن السيبراني عبر غرف تفاعلية (Rooms) تحتوي على سيناريوهات عملية. تستهدف المبتدئين وتوفر مسارات تعلم منظمة (Learning Paths).

نقاط القوة: بيئات عملية متقدمة، مسارات واضحة، مجتمع نشط. نقاط الضعف: باللغة الإنجليزية فقط، الخطة المجانية محدودة.

2.2.2 منصة HackTheBox

منصة متقدمة تستهدف المحترفين عبر آلات افتراضية (Machines) يجب اختراقها. تعتبر المنصة الأولى عالمياً لتحديات CTF الاحترافية.

نقاط القوة: سيناريوهات واقعية جداً، شهادات CPTS معتمدة. نقاط الضعف: غير مناسبة للمبتدئين، إنجليزية بالكامل، تكلفة مرتفعة.

2.4 تحليل المشكلة

من خلال دراسة الأنظمة الحالية والمسح الميداني الذي شمل عينة من 120 طالباً في تخصصات الحوسبة والأمن السيبراني في 3 جامعات سعودية، تم تحديد المشكلات الجوهرية التالية:

المشكلة الرئيسية: غياب منصة عربية متكاملة تجمع بين التعليم النظري والتطبيق العملي في الأمن السيبراني، مما يُجبر الطلاب على التنقل بين عدة منصات أجنبية بتكاليف مرتفعة ومحتوى لا يراعي السياق العربي.

المشكلات الفرعية:

  1. نقص المحتوى العربي المتقدم: 94% من المحتوى المتخصص باللغة الإنجليزية.
  2. ضعف التفاعل العملي: المنصات العربية تقتصر على الفيديوهات دون تجارب عملية.
  3. غياب المختبرات الافتراضية: لا توجد منصة عربية توفر بيئات اختراق/حماية تفاعلية.
  4. عدم ربط التعلم بسوق العمل: المحتوى المتوفر لا يتوافق مع متطلبات الشهادات المهنية.
  5. تكلفة التدريب المرتفعة: الاشتراكات في المنصات المتخصصة تتراوح بين 300-700 دولار سنوياً.

// Middleware: Authentication Guard
export async function getAuthUser(req: Request) {
  const token = req.headers.get('cookie')
    ?.split(';')
    .find(c => c.trim().startsWith('session='))
    ?.split('=')[1];
    
  if (!token) return null;
  
  const session = await db.session.findUnique({
    where: { token },
    include: { user: true },
  });
  
  if (!session || session.expiresAt < new Date()) return null;
  return session.user;
}
